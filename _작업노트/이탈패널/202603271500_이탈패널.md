## 확정된 개선 항목

### 1. 이탈범위 표시 버그 수정 ✅ 완료
**원인**: `DashboardPage.jsx` `applyOverride` 리턴 시 계산한 `yMin`/`yMax`를 슬롯 객체에 포함하지 않음
→ 패널이 가이드라인 임계값 대신 원래 센서 스펙(예: 10~40°C)으로 비교하여 이탈범위가 항상 null

**수정 내용** (`DashboardPage.jsx` line ~410):
```js
// Before
return { ...base, dataStatus: alertDataStatus, alertDelayMin }

// After
return {
  ...base,
  dataStatus: alertDataStatus,
  alertDelayMin,
  yMin: yMin ?? base.yMin,
  yMax: yMax ?? base.yMax,
}
```
> 가이드라인 값이 있으면 우선 사용, 없으면 원래 슬롯 값 유지

### 2. 카드 색상 정책 (DeviationPanel.css)

| 상태 | 조건 | 스타일 |
|---|---|---|
| **기본 (주황)** | default | `border-left: 4px solid var(--accent)` + 주황 배경 tint |
| **위험 (빨강)** | 오늘 누적 ≥ 10분 OR 기준범위 10% 초과 이탈 | `border-left: 4px solid var(--danger)` + 빨강 배경 tint |

**10% 이탈 계산식**:
```
이탈량 / (yMax - yMin) ≥ 0.1
```

**색상 토큰**:
- 주황: `var(--accent)` = `#E5401A`
- 빨강: `var(--danger)` = `#e05c5c`

### 4. 시간 표시 단위 (useDeviationTracker.js - formatDuration)

```
변경 전: "12분 30초", "45초"
변경 후: "12분", "1분 미만"  (초 단위 제거)
```

```js
export function formatDuration(ms) {
  if (!ms || ms < 60000) return '1분 미만'
  const totalMin = Math.floor(ms / 60000)
  const h = Math.floor(totalMin / 60)
  const m = totalMin % 60
  if (h > 0) return `${h}시간 ${m}분`
  return `${m}분`
}
```

## 오늘 누적 이탈 계산 정책 (확정)

```
todayAccumulatedMs = 종료된 이탈 누적(accumMs) + 현재 진행 중인 이탈(now - startMs)
```

**예시**:
- 09:00 이탈 시작 → startMs = 09:00
- 09:30 정상 복귀 → accumMs += 30분 (localStorage 저장)
- 09:33 재이탈 → startMs = 09:33
- 09:36 현재 → todayAccumulatedMs = 30분 + 3분 = **33분** ✓

**자정 초기화**: tr.date !== today 감지 시 accumMs = 0, 새 날짜 키로 재시작
**localStorage 키**: `deviation:{slotId}:{YYYY-MM-DD}`
