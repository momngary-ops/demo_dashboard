# 상단배너 KPI 슬롯 확정 — 202603251530

## 변경 개요
TopBanner KPI 카드 슬롯 2가지 개선:
1. localStorage 저장으로 새로고침 후에도 설정 유지
2. 기본 KPI 5개 항목 변경

---

## 1. localStorage 저장 (`TopBanner.jsx`)

### localStorage 키
```
topbanner:slots
```

### 초기 로드 (lazy initializer)
```jsx
const [slotConfigs, setSlotConfigs] = useState(() => {
  try {
    const raw = localStorage.getItem(STORAGE_KEY_BANNER_SLOTS)
    if (raw) return JSON.parse(raw)
  } catch { /* 무시 */ }
  return DEFAULT_SLOT_CONFIGS
})
```

### 저장 (항목 변경 시)
```jsx
onSlotsChange={(next) => {
  setSlotConfigs(next)
  try { localStorage.setItem(STORAGE_KEY_BANNER_SLOTS, JSON.stringify(next)) } catch { /* 무시 */ }
}}
```

### 동작
- 최초 방문 또는 localStorage 없음 → `DEFAULT_SLOT_CONFIGS` 사용
- "항목 추가/변경" 모달로 변경 후 저장 → `topbanner:slots`에 전체 슬롯 객체 JSON 저장
- 새로고침 후에도 변경된 항목 유지

### 기존 설정 초기화 방법
```js
// 브라우저 콘솔에서 실행
localStorage.removeItem('topbanner:slots')
```

## 2. 기본 KPI 5개 변경 (`kpiCandidates.js`)

### DEFAULT_SLOT_CONFIGS

| 순서 | ID | 타이틀 | 단위 | 이전 |
|---|---|---|---|---|
| 1 | `xintemp1` | 내부온도 | °C | 유지 |
| 2 | `xinhum1` | 내부습도 | % | 유지 |
| 3 | `xco2` | CO₂ 농도 | ppm | 유지 |
| 4 | `xinsunadd` | 누적 일사량 | J/cm² | 급액 EC (now_ec) |
| 5 | `xhumlack` | VPD | g/m³ | 함수율 (water_con) |

모든 항목은 `ENV` 배열에서 조회. (`xinsunadd`, `xhumlack` 모두 ENV 소속)
