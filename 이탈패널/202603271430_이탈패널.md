## 기능 상세

### DeviationPanel
- OUT_OF_RANGE 슬롯이 1개 이상일 때 TopBanner 아래, 위젯 그리드 위에 표시
- 4열 CSS Grid (반응형: ≤1199px → 3열, ≤767px → 2열, ≤479px → 1열)
- 헤더: "⚠ 임계치 이탈 센서 N건" + "모두 닫기" 버튼

### DeviationCard (카드 1개 구성)
- 헤더: 센서 아이콘 + 이름 + 구역 + ⚠ 배지
- 2×2 바디 그리드:
  - 현재값: 실시간 측정값 + 단위
  - 이탈 범위: 초과/미달 수치 + 상한/하한 기준값 (`yMin`/`yMax` 기반)
  - 이탈 지속시간: 이탈 시작~현재 (1초 갱신)
  - 오늘 누적 이탈: 당일 00:00 기준 누적 ms (localStorage 영속)

### 시각 효과
- 이탈 중: `--danger` 좌측 보더 3px + 배경 tint
- 이탈 30분 이상: `dev-pulse` 애니메이션 (느린 보더 색상 변화)

## 표시/숨김 정책

```
표시: deviatedSlots.length > 0 && dismissedAt === null

닫기: setDismissedAt(Date.now())
  → 이탈 슬롯이 남아있어도 패널 완전 소멸

재표시: notifications 배열에서 최신 OUT_OF_RANGE 알림의
        timestamp > dismissedAt → setDismissedAt(null)
  (딜레이 경과 후 새 알림이 발송된 시점에만 재등장)
```

## useDeviationTracker 동작

```
입력: alertSlots[] (전체)
출력: Map<slotId, { currentElapsedMs, todayAccumulatedMs }>

- prevRef로 이전 dataStatus 추적 → 전환 감지
- 정상 → OUT_OF_RANGE: startMs = Date.now()
- OUT_OF_RANGE → 정상: accumMs += (now - startMs), localStorage 저장
- setInterval 1000ms: OUT_OF_RANGE 슬롯 있을 때 리렌더 트리거
- 자정 초기화: tr.date !== today 감지 시 accumMs = 0, 새 날짜 키로 재시작
- localStorage 키: deviation:{slotId}:{YYYY-MM-DD}
```

## 가이드라인 연동 상태

DashboardPage.jsx의 `applyOverride` 함수에서 이미 올바르게 구현됨 (확인):
- `ac?.delay_min` ✓
- `(ac?.deviation_pct ?? 10) / 100` ✓
- CO₂: `ref * (1 ± devRate)` 계산 ✓
- 판정 보류 조건 (enabled=false, guidelines null, 0값, min>max) ✓
