# 스마트팜 관제 대시보드 — 프로젝트 개요

> 작성일: 2026-03-17

---

## 1. 목적

스마트팜 통합 관제를 위한 **반응형 커스터마이징 대시보드** 구현.
- 농장 환경 데이터를 한 눈에 파악 (Glanceability)
- 사용자가 직접 위젯 배치·크기·항목을 조정 가능

---

## 2. 기술 스택

| 항목 | 선택 |
|---|---|
| 프레임워크 | React 18 + Vite |
| 언어 | JavaScript (JSX) — TypeScript 미사용 |
| 스타일 | 순수 CSS + CSS 변수 (다크 테마) — Tailwind 미사용 |
| 그리드 | react-grid-layout **v2.2.2 / legacy API** |
| 아이콘 | lucide-react |
| 차트 | 순수 SVG 스파크라인 (외부 라이브러리 없음) |

---

## 3. 전체 파일 구조

```
src/
├── index.css                    # 전역 CSS 변수 (다크 테마)
├── App.jsx / App.css            # 메인 레이아웃 (사이드바 + 헤더 + 콘텐츠)
├── components/
│   ├── Sidebar.jsx/.css         # 접이식 사이드바 (200px ↔ 56px)
│   ├── Header.jsx/.css          # 상단 헤더 (56px 고정)
│   ├── Widget.jsx/.css          # 그리드 위젯 컴포넌트
│   ├── WidgetPicker.jsx/.css    # 위젯 추가 모달
│   └── TopBanner/               # 헤드 위젯 (상세: 헤드위젯/ 문서 참고)
├── pages/
│   └── DashboardPage.jsx/.css   # 대시보드 페이지 (그리드 + 헤드위젯 통합)
├── constants/
│   ├── kpiCandidates.js         # 15개 KPI 후보 풀
│   └── pollingConfig.js         # 폴링 주기 상수
├── hooks/
│   ├── useKpiPolling.js         # KPI 데이터 폴링
│   └── useWeatherPolling.js     # 날씨 폴링
├── utils/
│   └── kpiStatusResolver.js     # KPI 상태 판정 로직 (9가지)
└── mocks/
    ├── kpiMockData.js           # KPI 목 데이터
    └── weatherMockData.js       # 날씨 목 데이터
```

