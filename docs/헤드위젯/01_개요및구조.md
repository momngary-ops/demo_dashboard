# 헤드 위젯 (TopBanner) — 개요 및 구조

> 작성일: 2026-03-17

---

## 1. 목적

대시보드 상단에 위치하는 **KPI 요약 배너**.
- 1초 내 농장 핵심 상태 파악 (Glanceability)
- 현재 외부 날씨에 따라 배경이 동적으로 변경
- 구역 탭 전환으로 시설 구역별 현황 확인


## 5. 폴링 아키텍처

### 현재 상태: Mock 모드

실제 API 연동 준비 완료 — 훅 내부만 교체하면 나머지 컴포넌트 변경 없음.

```
useKpiPolling(slotConfigs)
  → 현재: MOCK_API_DATA에서 즉시 반환 (800ms 딜레이 시연)
  → 실제: setInterval(30s) + Promise.all([fetch(slot1), ..., fetch(slot5)])

useWeatherPolling()
  → 현재: MOCK_WEATHER 상수 반환
  → 실제: setInterval(5min) + fetch(기상청 API)
```

### 폴링 상수 (constants/pollingConfig.js)

| 상수 | 값 | 설명 |
|---|---|---|
| `KPI_INTERVAL_MS` | 30,000ms | KPI 갱신 주기 |
| `WEATHER_INTERVAL_MS` | 300,000ms | 날씨 갱신 주기 (5분) |
| `REQUEST_TIMEOUT_MS` | 3,000ms | 단일 요청 타임아웃 |
| `STALE_WARN_MS` | 300,000ms | 5분 초과 → 주황 경고 |
| `STALE_CRIT_MS` | 600,000ms | 10분 초과 → 빨강 + 흐림 |
