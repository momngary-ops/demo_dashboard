# 헤드 위젯 — 날씨 배경 시스템 (WeatherBackground)

> 작성일: 2026-03-17

---

## 1. 개요

`weather_condition` 값에 따라 TopBanner 배경 그라데이션 + 애니메이션이 동적으로 변경됨.
- 배경 전환: CSS `transition: background 1s ease-in-out`
- API 실패 시 fallback: `CLEAR` (절대 크래시 금지)

---

## 2. 날씨 조건별 배경 스펙

| condition | 한국어 | 그라데이션 | 애니메이션 |
|---|---|---|---|
| `CLEAR` | 맑음 | `#38bdf8 → #7dd3fc → #bae6fd` | 없음 (태양 제거됨) |
| `PARTLY_CLOUDY` | 구름 조금 | `#60a5fa → #93c5fd → #cbd5e1` | 구름 2개 float |
| `CLOUDY` | 흐림 | `#64748b → #94a3b8 → #cbd5e1` | 구름 4개 float |
| `RAIN` | 비 | `#1e3a5f → #334155 → #475569` | 빗줄기 10개 |
| `HEAVY_RAIN` | 폭우 | `#0f172a → #1e293b → #334155` | 빗줄기 18개 |
| `SNOW` | 눈 | `#e0f2fe → #f0f9ff → #ffffff` | 눈송이 14개 |
| `NIGHT_CLEAR` | 맑은 밤 | `#0f172a → #1e1b4b → #312e81` | 별 22개 반짝임 |
| `NIGHT_RAIN` | 밤비 | `#0c0a1e → #1e1b4b → #0f172a` | 빗줄기 10개 |

---

## 3. 애니메이션 구현

모든 애니메이션은 CSS `@keyframes` 기반. 결정론적 좌표(Math.random 미사용)로 리렌더링 안전.

### 구름 (CloudAnim)
```css
@keyframes cloudFloat {
  0%, 100% { transform: translateX(0); }
  50%       { transform: translateX(10px); }
}
```
- 2~4개 SVG 구름, 각각 `animationDelay` 다르게 설정

### 빗줄기 (RainAnim)
```css
@keyframes rainFall {
  from { transform: translateY(-10px); opacity: 0; }
  10%  { opacity: 1; }
  to   { transform: translateY(220px); opacity: 0; }
}
```
- `.wb-raindrop`: 1.5px 너비 반투명 파란 선
- heavy: 18개 / normal: 10개

### 눈 (SnowAnim)
- ❄ 문자 14개, 각각 다른 속도·딜레이로 낙하

### 별 (StarsAnim)
```css
@keyframes starTwinkle {
  0%, 100% { opacity: 0.15; transform: scale(0.8); }
  50%       { opacity: 1;    transform: scale(1.2); }
}
```
- 흰색 원형 div 22개, 위치·딜레이 결정론적 계산

---

## 4. 레이어 구조 (z-index)

```
.topbanner__bg      (absolute, 배경 그라데이션)
.topbanner__overlay (absolute, rgba(0,0,0,0.18) 가독성 오버레이)
.wb-anim            (absolute, pointer-events:none, 애니메이션 요소들)
.topbanner__content (relative, z-index:1, 실제 콘텐츠)
```

---

## 5. 텍스트 가독성

모든 날씨 조건에서 흰색 텍스트 사용.
`rgba(0,0,0,0.18)` 오버레이 + `text-shadow`로 밝은 배경(CLEAR, SNOW)에서도 가독성 확보.
