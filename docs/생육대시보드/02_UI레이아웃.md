# 생육 대시보드 — UI 레이아웃

> 작성일: 2026-03-17

---

## 1. 전체 구조

메인 대시보드와 **동일한 레이아웃 시스템** 사용.

```
┌─────────────────────────────────────────────────────────────┐
│ Sidebar (200px)     │ Header (56px)                         │
│ ─────────────────   ├───────────────────────────────────────│
│ 대동 / Smart파밍    │ 페이지 툴바                            │
│                     │  데이터 입력  [날짜] [구역▼] [로봇] [활성화]│
│ □ Dashboard         ├───────────────────────────────────────│
│ □ 대시보드          │                                        │
│ □ AI 레포트         │  입력 카드 그리드 (11개 항목)           │
│ □ 데이터 조회       │                                        │
│ ■ 데이터 입력 ◄     │                                        │
│                     ├───────────────────────────────────────│
│ Account             │  n개 완료 / 총 11개   [데이터 저장하기] │
│ □ Our community     │                                        │
│ □ Profile           │                                        │
│ □ Contact Us        │                                        │
│ □ Logout            │                                        │
└─────────────────────────────────────────────────────────────┘
```

## 2. CSS 변수 (index.css 공유)

| 변수 | 값 | 용도 |
|---|---|---|
| `--bg` | `#0d1520` | 최상위 배경 |
| `--sidebar-bg` | `#101d2b` | 사이드바·헤더 배경 |
| `--sidebar-active` | `#1a2d42` | 활성 메뉴 배경 |
| `--surface` | `#131f30` | 카드 배경 |
| `--surface-2` | `#192840` | 입력 필드 배경 |
| `--border` | `#1c2e42` | 테두리 |
| `--text-primary` | `#dde8f0` | 본문 텍스트 |
| `--text-secondary` | `#6e93b0` | 보조 텍스트 |
| `--text-muted` | `#3f607a` | 흐린 텍스트 |
| `--accent` | `#2d7dd2` | 포인트 색상 (사이드바 로고 등) |
| `--danger` | `#e05c5c` | 활성화·저장 버튼, 오류 |
| `--robot-green` | `#6AC8C7` | 로봇 수집 데이터 표시 (대동 컬러) |

---

## 3. 페이지 툴바

```
[데이터 입력]                    [날짜] [구역▼] │ [로봇 데이터] [입력 활성화]
     ↑ page__title                                        ↑ toolbar-controls
```

### 상태별 버튼 변화

| 상태 | 버튼 |
|---|---|
| 비활성 | 빨간 `입력 활성화` 버튼 |
| 활성 후 | 초록 펄스 + `n구역 입력 중 (x/11)` |
| 로봇 데이터 | 비활성 시 disabled, 활성 시 클릭 가능 |

---

## 4. 입력 카드

```
┌─────────────────────────┐
│ 초장 (XX.X)   ✓  🤖     │  ← 카드 헤더: 항목명 + 포맷 + 유효성 아이콘 + 출처 배지
├─────────────────────────┤
│  [32] . [5]        cm   │  ← 입력 영역 + 단위
└─────────────────────────┘
```

### 카드 상태별 테두리

| 상태 | 테두리 색 |
|---|---|
| 미입력 | `--border` (기본) |
| 입력 완료 | `--robot-green` (#6AC8C7) + ✓ |
| 미완료(일부) | `--danger` (#e05c5c) + ✗ |

### 출처 배지

| 아이콘 | 의미 |
|---|---|
| 🤖 Bot (초록) | 로봇 API 수집 데이터 |
| 👤 User (회색) | 수기 입력 데이터 |

---

## 5. 하단 바

- 좌측: `n개 완료 / 총 11개` (활성 상태에서만 표시)
- 우측: `데이터 저장하기` 버튼 (`--danger` 색, 비활성 시 disabled)
