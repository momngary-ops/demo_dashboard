## 2. 입력 타입

### select (드롭다운)
- 범위가 명확한 항목에 사용
- 오입력 방지 목적
- `generateOptions(min, max)` 함수로 옵션 자동 생성

### input (자유 입력)
- `type="number"`
- 범위 제약이 없거나 소수점 자유 입력 필요 항목

### 복합 입력 (prefix `.`)
- 정수부 + `.` + 소수부를 분리 입력
- 예: 초장 `32` + `.` + `5` → `32.5 cm`

## 3. 유효성 검증 로직

```js
// 해당 항목의 모든 input이 '' 아닌 값을 가질 때 valid
const isFilled = newData[fieldId].every(val => val !== '');
setFormValid(prev => ({ ...prev, [fieldId]: isFilled }));
```

| formValid 값 | 의미 |
|---|---|
| `null` | 아직 터치하지 않음 |
| `true` | 모든 입력 완료 |
| `false` | 일부 입력 후 미완료 |
