## 3. react-grid-layout 주의사항

### ⚠️ v2.2.2 API 변경

- **기존 (v1 스타일)** props가 v2.2.2 `GridLayout` (default export)에서 무시됨
- **해결**: `react-grid-layout/legacy` import 사용

```js
// ❌ 잘못된 방식 (v2.2.2 default에서 cols/resizeHandles 등 무시됨)
import GridLayout from 'react-grid-layout'

// ✅ 올바른 방식
import GridLayout from 'react-grid-layout/legacy'
```

- `react-grid-layout/legacy`는 구형 prop API를 새 API(`resizeConfig`, `dragConfig` 등)로 변환해주는 래퍼

## 4. 리사이즈 핸들 (6방향)

```jsx
resizeHandles={['se', 'sw', 'ne', 'nw', 's', 'e']}
draggableHandle=".widget__drag-handle"
```

### CSS 핵심 원칙

- react-resizable은 `background-image: url(SVG)` + `transform: rotate()` 방식으로 핸들 방향 지정
- **`transform`을 직접 오버라이드 금지** → 회전 방향이 깨짐
- **`::after` 규칙 금지** → react-resizable은 `::after`가 아닌 background-image 사용

```css
/* ✅ 올바른 override */
.react-resizable-handle {
  background-image: url("data:image/svg+xml,...") !important; /* accent 색상 */
  opacity: 0;
  transition: opacity 0.15s;
}
.react-grid-item:hover .react-resizable-handle { opacity: 1; }
```

## 5. 그리드 오버레이 (편집모드)

편집 모드 진입 시 SVG 선으로 컬럼·행 가이드라인 표시.

- 컬럼 경계선: 21개 수직선 (`COLS + 1`)
- 행 경계선: 13개 수평선
- 컬럼 번호 표시 (1~20)
- 색상: `rgba(45, 125, 210, 0.18)` (accent 반투명)
- 위젯은 `z-index: 1`로 오버레이 위에 렌더링

---

## 6. 반응형 대응

사이드바 접힘/펼침 시 그리드 너비 자동 조정:

```jsx
const containerRef = useCallback(node => {
  if (!node) return
  const ro = new ResizeObserver(([entry]) => {
    setContainerWidth(entry.contentRect.width)
  })
  ro.observe(node)
}, [])
```

`containerWidth`를 `GridLayout`의 `width` prop으로 전달 → 그리드가 실시간으로 재계산됨.
