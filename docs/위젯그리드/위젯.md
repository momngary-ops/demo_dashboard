# 위젯 그리드 — 위젯 컴포넌트 시스템

> 작성일: 2026-03-17

---

## 1. 위젯 타입

| type | 컴포넌트 | 설명 |
|---|---|---|
| `stat` | `StatWidget` | 단일 수치 표시 (값 + 단위 + 증감) |
| `chart` | `ChartWidget` | 차트 플레이스홀더 (API 연동 후 표시) |

---

## 2. Widget.jsx 구조

```
┌─────────────────────────────────┐
│ ≡ 제목              W×H  [×]   │  ← 헤더 (편집모드 시 표시 요소 추가)
├─────────────────────────────────┤
│                                 │
│      (StatWidget / ChartWidget) │  ← 바디
│                                 │
└─────────────────────────────────┘
```

### 편집 모드 추가 요소
- `≡` 드래그 핸들 (`.widget__drag-handle`) — lucide GripHorizontal
- `W×H` 크기 배지 (`.widget__size-badge`) — 현재 grid 크기 실시간 표시
- `×` 제거 버튼 (`.widget__remove`)

---

## 3. Props

```jsx
<Widget
  id={i}              // 위젯 고유 ID
  config={widgets[i]} // { type, title, dataKey }
  editMode={editMode} // 편집 모드 여부
  onRemove={fn}       // 위젯 제거 콜백
  gridSize={sizeMap[i]} // { w, h } 현재 그리드 크기
  isResizing={bool}   // 리사이즈 중 여부
/>
```

## 4. Stat 위젯 Mock 데이터

```js
const MOCK = {
  temperature: { value: '24.3', unit: '°C',  delta: '+0.5', up: true },
  humidity:    { value: '68',   unit: '%',   delta: '-2.1', up: false },
  co2:         { value: '412',  unit: 'ppm', delta: '+8',   up: true },
  light:       { value: '3,200',unit: 'lx',  delta: '+120', up: true },
}
```

- `up: true` → 초록 ▲ delta
- `up: false` → 빨강 ▼ delta

---

## 5. 위젯 추가 (WidgetPicker)

- 편집 모드에서 `+ 위젯 추가` 버튼 클릭
- 모달에서 위젯 타입 선택
- 새 위젯: `x:0, y: maxY, w:5, h:5, minW:2, minH:2`로 추가

---

## 6. 최소 크기 제약

| 항목 | 값 |
|---|---|
| `minW` | 2 (grid column) |
| `minH` | 2 (grid row) |

리사이즈 시 이 값 미만으로 줄어들지 않음.
