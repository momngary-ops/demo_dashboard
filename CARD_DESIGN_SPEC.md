## 공통 컬러 시스템 — 신호등 방식

> 전체 UI는 다크 테마에서 크게 벗어나지 않는 수준 유지.
> 색상은 배경에 사용하지 않고 **신호등 dot + 텍스트 색**으로만 표현.

### 신호등 dot 스펙

```
크기:   8px 원형 (● )
위치:   row1 value 왼쪽
```

| 상태 | dot 색 | 텍스트 색 | 조건 |
|------|--------|----------|------|
| **미설정** | `#3f607a` (회색) | `--text-secondary` | 가이드라인 없음 (기본) |
| **정상** | `#4ade80` (초록) | `--text-primary` | 가이드라인 범위 내 |
| **주의** | `#fb923c` (주황) | `#fb923c` | warn 임계 초과 |
| **위험** | `#f87171` (빨강) | `#f87171` | crit 임계 초과 |

```
● 회색  →  임계값 미설정, 가이드라인 없음
● 초록  →  정상 범위
● 주황  →  주의 (warn)
● 빨강  →  위험 (crit)
```

### 적용 범위

- `chart-main` row1 value 앞에만 dot 표시
- `stat` 카드: dot 미사용 (단순 참고값, 가이드라인 별도 적용 시 추가)
- `multi-line`: 위험 구간 배경 하이라이트 (dot 아닌 영역 강조)
- `gauge-set` / `status-panel`: 자체 색상 체계 사용 (별도 정의)

### 현재값 글자 크기

```
타이틀:   16px  semi-bold  --text-primary
row1 값:  20px  bold       신호등 색 적용
row1 단위: 13px normal     --text-secondary
row2 값:  13px  normal     --text-secondary
delta:    12px  normal     trend-up / trend-down
trend:    ▲▼   12px       동일
```

```

```

#### 곡선 처리 — Catmull-Rom → Cubic Bezier 변환

꺾임 없는 라운딩을 위해 직선(L) 대신 **3차 베지어 곡선(C)** 사용.
각 포인트 사이를 Catmull-Rom 알고리즘으로 제어점을 계산하여 자연스러운 곡선 생성.

```
tension = 0.3   ← 낮을수록 완만, 높을수록 데이터에 밀착

각 세그먼트 P[i] → P[i+1]:
  cp1x = P[i].x   + (P[i+1].x - P[i-1].x) * tension / 2
  cp1y = P[i].y   + (P[i+1].y - P[i-1].y) * tension / 2
  cp2x = P[i+1].x - (P[i+2].x - P[i].x)   * tension / 2
  cp2y = P[i+1].y - (P[i+2].y - P[i].y)   * tension / 2

SVG path: M x0,y0  C cp1x,cp1y cp2x,cp2y x1,y1  C ...
```

> SVG 직접 구현 위젯(chart-main 스파크라인) 전체에 동일 적용.
> 데이터 포인트가 각져도 렌더링은 항상 라운딩 처리.

### 상태별 처리

| 상태 | 표현 |
|------|------|
| LOADING | 스파크라인 자리: pulse shimmer · 수치: `--` |

## 타이틀 인라인 편집

### 정책

```
kpiId (API 연결)  →  변경 불가 (고정)
title (표시 이름) →  편집 가능 (위젯별 독립 저장)
```

> 농장마다 현장에서 부르는 이름이 다를 수 있으므로 표시명은 자유롭게 수정 가능.
> API 매핑(kpiId)은 고정이므로 데이터 연결에는 영향 없음.

```
  타이틀: [ 외부온도 ✎ ]  ← 밑줄 + 연필 아이콘으로 편집 가능 표시
  클릭 →  [ 외부온도       ]  ← input 필드로 전환 (기존 값 선택된 상태)
  Enter or 외부 클릭 →  저장 후 input → 텍스트로 복귀
  ESC →   변경 취소
```

---

### 시각 처리

| 상태 | 타이틀 스타일 |
|------|--------------|
| 편집모드 OFF | 텍스트, 커서 default |
| 편집모드 ON (hover) | 밑줄 점선 `border-bottom: 1px dashed --text-muted` + ✎ 아이콘 |
| 편집 중 (input) | input 박스, `background: --surface-2`, 테두리 `--accent`, 같은 폰트·크기 유지 |

```
편집모드 ON, hover 시:
  외부온도 ✎
  ─ ─ ─ ─ ─

편집 중:
  ┌────────────────┐
  │ 외부온도       │  ← input, --accent border
  └────────────────┘
```
