        이 세 값이 한 차트에 표시될 때 제어 루프 전체를 이해할 수 있음.

[ 046 ] 순환펌프 작동상태 | xpumprun1 | 0: 정지 / 1: 기동
  판정: ✅ '장비 가동 현황 패널' + 난방 차트 오버레이
  의견: 034(순환펌프 자수동)의 실제 가동 상태 쌍. 패널 구성 항목.
        018·019(공급·회수온도) 차트에 펌프 가동 구간 오버레이 시
        온수 순환 여부와 난방 효율 연관성 파악 가능.
        가동 현황 패널 마지막 구성 항목.

        ※ '장비 가동 현황 패널' 최종 구성 (035~038 + 043 + 045 + 046):
           CO₂공급(035) / 보광등(036) / 훈증기(037) / 보일러(038) /
           보조기기(043) / 냉난방기(045) / 순환펌프(046)

        ※ '설정 상태 패널(자수동)' 최종 구성 (027~034 + 042 + 044):
           창문(027) / 커튼(028) / 3Way(029) / CO₂(030) /
           보광등(031) / 훈증기(032) / 보일러(033) / 순환펌프(034) /
           보조기기(042) / 냉난방기(044)

[ 047 ] 설정량 | set_val1 | 단위 확인 必
  판정: ✅ '양액 관수 현황' 3종 세트 구성 (048·049와 묶음)
  의견: 관수 1회당 목표 공급량. 단위 미확정(PDF에도 "단위 확인 必" 명시).
        048(잔여량), 049(누적량)과 함께 '양액 관수 현황' 세트 구성.
        단독 꺾은선보다 세트 위젯이 훨씬 유용.
        ※ 단위 확정 후 위젯 라벨 업데이트 필요.

[ 048 ] 잔여량 | re_val1 | 단위 확인 必
  판정: ✅ 꺾은선 위젯 (저수위 경보 임계값 표시 권장)
  의견: 양액 탱크 잔여량.
        관수 진행 시 감소 → 보충 시 급상승하는 톱니 패턴.
        꺾은선으로 잘 표현됨.
        저수위 경보 임계값 라인 함께 표시 권장.
        047·049와 '양액 관수 현황' 세트로 묶음.

[ 049 ] 누적량 | ac_val1 | 단위 확인 必
  판정: ✅ 게이지/진행바 위젯 권장 (꺾은선도 가능)
  의견: 당일 누적 관수량. 007·009(누적일사량)과 유사하게 하루 단조 증가, 일 단위 리셋.
        관수 횟수 및 총량 관리 지표.
        목표 설정량(047) 대비 현재 누적량을 게이지/진행바로 표시 시 직관적.
        예: 목표 1000L 대비 현재 340L 진행 (34%)
        047·048과 '양액 관수 현황' 세트로 묶음.
