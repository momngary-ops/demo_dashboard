[ 021 ] 내부습도 | xinhum1 | %
  판정: ✅ 꺾은선 위젯 최우선 (TopBanner KPI 카드 이미 구현)
  의견: 온실 환경의 핵심 지표 중 하나. KPI 카드 구현 완료.
        위젯으로 추가 시 이슬점(017)과 함께 멀티라인 차트로 표시 권장.
        내부습도가 이슬점에 근접(포화 상태)하면 결로·병해 위험 구간 시각화 가능.
        현장 중요도 최상.

[ 022 ] CO₂농도 | xco2 | ppm
  판정: ✅ 꺾은선 위젯 최우선 (TopBanner KPI 카드 이미 구현)
  의견: 광합성 관리의 핵심 데이터. KPI 카드 구현 완료.
        꺾은선 위젯에서 CO₂ 보충기 작동(035) 시점이 값 급상승으로 명확히 표현됨.
        023(CO₂설정값)과 반드시 멀티라인 쌍으로 구성 권장.
        주간 보충 → 야간 감소 패턴도 식물 호흡 이해에 유용.

[ 023 ] CO₂설정값 | xco2set | ppm
  판정: ⚠️ 단독 위젯 불가 — 022와 멀티라인 쌍 필수
  의견: 단독으로는 의미 없음.
        022(실제 CO₂농도)와 쌍으로 표시해야 제어 성능 확인 가능.
        실제값이 설정값을 얼마나 잘 추적하는지 → CO₂ 공급 시스템 효율 판단.
        패턴: 020(내부온도) + 040(환기설정) + 041(난방설정) 온도 세트와 동일 구조.

[ 024 ] 창문개도 | xwinvol1_1
  판정: ✅ 게이지/현재값 위젯 (025, 026과 구동기 개도 세트 묶음 권장)
  의견: 환기창 개도 상태. 내부온도·외부온도 상승 시 창이 열리는 상관관계를
        꺾은선으로 확인 가능.
        단독보다 025(커튼개도), 026(3Way밸브개도)과 함께
        '구동기 개도 현황' 세트 위젯으로 구성 시 직관성 높음.
        게이지 바 또는 현재값 + 추이 형태 권장.

[ 025 ] 커튼개도 | xcur1vol | %
  판정: ✅ 게이지/현재값 위젯 (024, 026과 구동기 개도 세트 묶음 권장)
  의견: 차광·보온 커튼의 개도 비율.
        외부일사량(006)·내부일사량(008)과 연동해 자동 제어 패턴 확인 가능.
        024·026과 함께 '구동기 개도 현황' 세트로 구성.

[ 026 ] 3Way밸브개도 | x3way1vol | %
  판정: ✅ 게이지/현재값 위젯 (024, 025와 구동기 개도 세트 묶음 권장)
  의견: 난방 회로 3방향 밸브 개도.
        012·018·019 난방 온도 세트와 함께 표시하면
        난방 제어 동작 전체(온도 → 밸브 반응)를 한눈에 파악 가능.
        024·025와 '구동기 개도 현황' 세트로 묶거나
        난방 세트(012·018·019)와 묶는 방식 모두 유효. 설계 시 결정 필요.

[ 027 ] 창 자수동 | xwin1auto | 자동/수동
  판정: ⚠️ 독립 위젯 불필요 — 027~034 통합 상태 패널 위젯으로 처리
  의견: 이진 상태값. 꺾은선·게이지 모두 의미 없음.
        수동 전환 시 자동화 제어 무력화 → 경보 역할이 핵심.
        028·029·030 (및 031~034)과 함께 '시스템 자동화 상태' 패널 위젯으로 통합 권장.
        각 항목을 컬러 뱃지(자동=초록 / 수동=주황) 형태로 한 위젯에 나열.

[ 028 ] 커튼 자수동 | xcur1auto | 자동/수동
  판정: ⚠️ 독립 위젯 불필요 — 통합 상태 패널로 처리 (027 참조)
  의견: 027과 동일 패턴. 커튼이 수동 상태이면 일사량 대응 자동 차광 불가.
        통합 패널에서 한 줄로 표시.

[ 029 ] 3Way 자수동 | x3way1auto | 자동/수동
  판정: ⚠️ 독립 위젯 불필요 — 통합 상태 패널로 처리 (027 참조)
  의견: 027·028과 동일. 3Way 밸브 수동 시 난방 자동 제어 불가.
        통합 패널에 포함.

[ 030 ] CO₂ 자수동 | xco2auto | 자동/수동
  판정: ⚠️ 독립 위젯 불필요 — 통합 상태 패널로 처리 (027 참조)
  의견: CO₂ 공급기가 수동 상태이면 농도 이상에도 자동 보충 안 됨.
        경보 가치 있음 → 통합 패널에서 수동 상태 시 강조 표시(빨간 뱃지) 권장.
