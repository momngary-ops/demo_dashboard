[ 031 ] 보광등 자수동 | xlightauto | 자동/수동
  판정: ⚠️ 독립 위젯 불필요 — 027~034 통합 '설정 상태 패널'로 처리
  의견: 보광등 자동/수동 제어 설정.
        수동 전환 시 일조 부족에도 자동 점등 불가.
        027~034 자수동 8종 통합 패널에 포함.

[ 032 ] 훈증기 자수동 | xhunauto | 자동/수동
  판정: ⚠️ 독립 위젯 불필요 — 027~034 통합 '설정 상태 패널'로 처리
  의견: 병해충 방제 훈증기 자동/수동 설정.
        수동 시 자동 방제 루틴 미작동.
        실시간 상태보다 작동 이력(037)과 연계 추적이 더 중요.
        통합 패널에 포함.

[ 033 ] 보일러 자수동 | xboauto | 자동/수동
  판정: ⚠️ 독립 위젯 불필요 — 통합 패널 내 경보 강조 필수
  의견: 027~034 중 경보 중요도 최상.
        수동 전환 시 야간 저온에서 자동 난방 불가 → 냉해 위험 직결.
        통합 패널에서 수동 상태 시 빨간 강조 표시 필수.

[ 034 ] 순환펌프 자수동 | xpumpauto | 자동/수동
  판정: ⚠️ 독립 위젯 불필요 — 027~034 통합 '설정 상태 패널' 완성
  의견: 027~034 자수동 8종 세트 마지막 항목.
        펌프 수동 시 온수 순환 중단 → 난방 효율 급락.
        통합 패널 완성. 총 8개 항목(027~034)이 하나의 위젯에 표시됨.

        ※ '설정 상태 패널' 위젯 구성 (027~034):
           창문(027) / 커튼(028) / 3Way(029) / CO₂(030)
           보광등(031) / 훈증기(032) / 보일러(033) / 순환펌프(034)
           → 각 항목: 자동=초록 뱃지 / 수동=주황 뱃지 / 수동(경보)=빨간 뱃지

--------------------------------------------------------------------------------
※ 이 지점에서 중요한 구분:
   027~034 = '자수동' → 제어 설정 상태 (자동 or 수동 오버라이드)
   035~038 = '작동상태' → 실제 장비 가동 여부 (켜짐/꺼짐)
   → 두 패널을 별도로 구성하는 것이 혼동 방지에 효과적.
--------------------------------------------------------------------------------

[ 035 ] CO₂ 작동상태 | xco2run | 0: 정지 / 1: 기동
  판정: ✅ '장비 가동 현황' 패널 + 022(CO₂농도) 차트 오버레이
  의견: CO₂ 주입기 실제 가동 여부.
        022(CO₂농도) 꺾은선 차트에 가동 구간을 배경 밴드로 오버레이하면
        CO₂ 주입 → 농도 상승 상관관계를 직관적으로 확인 가능.
        035~038 묶어 '장비 가동 현황' 패널 위젯으로 별도 구성 강력 권장.

[ 036 ] 보광등 작동상태 | xlightrun | 0: 정지 / 1: 기동
  판정: ✅ '장비 가동 현황' 패널 + 008(내부일사량) 차트 오버레이
  의견: 보광등 실제 점등 여부.
        DLI(Daily Light Integral) 관리에 직결.
        008(내부일사량) 차트에 점등 구간 오버레이 시
        자연광 + 보광 합산 광량 파악 가능.
        '장비 가동 현황' 패널에 포함.

[ 037 ] 훈증기 작동상태 | xhunrun | 0: 정지 / 1: 기동
  판정: ✅ '장비 가동 현황' 패널 + 이벤트 로그 연계 권장
  의견: 훈증(방제) 실행 중 여부. 평시 0, 방제 시에만 1로 전환.
        연속 시계열 차트보다 이벤트 발생 타임라인 기록이 더 적합.
        '장비 가동 현황' 패널에 포함하되 작동 이력 로그와 연계 권장.

[ 038 ] 보일러 작동상태 | xborun | 0: 정지 / 1: 기동
  판정: ✅ '장비 가동 현황' 패널 + 020(내부온도) 차트 오버레이
  의견: 보일러 실제 가동 여부.
        020(내부온도) 꺾은선 차트에 가동 구간 배경 밴드 오버레이 시
        난방 사이클(온도 하강 → 보일러 기동 → 온도 상승)이 한눈에 표현됨.
        '장비 가동 현황' 패널 포함. 현장 중요도 높음.

        ※ '장비 가동 현황' 패널 위젯 구성 (035~038):
           CO₂공급(035) / 보광등(036) / 훈증기(037) / 보일러(038)
           → 각 항목: 가동중=초록(점멸 가능) / 정지=회색

[ 039 ] 주야간상태 | xjuya | 주간: 1 / 야간: 0
  판정: ⚠️ 독립 위젯 불필요 — 차트 배경 음영 기준값으로 활용
  의견: 모든 시계열 차트의 야간 구간 배경 음영 처리 기준으로 활용 권장.
        야간 구간을 약간 어둡게 음영 처리하면 주야간 패턴 구분이 직관적.
        TopBanner 날씨 배경 연동 후보 (야간 → 별/달 배경 전환).
        독립 위젯으로는 의미 없음.

[ 040 ] 환기온도 | xventtemp1 | ℃
  판정: ⚠️ 단독 위젯 불가 — 020·041과 멀티라인 세트 필수
  의견: 환기창이 열리기 시작하는 설정 온도.
        020(내부온도) + 040(환기설정온도) + 041(난방설정온도) 멀티라인 세트의 구성 요소.
        실제 내부온도가 두 설정값 사이에서 제어되는 과정을 시각화.
        단독 표시 시 맥락 없어 의미 불명확.

================================================================================
종합 요약
================================================================================

  027~034 통합 완성: '설정 상태 패널' (자수동 8종)
    창문(027) / 커튼(028) / 3Way(029) / CO₂(030) /
    보광등(031) / 훈증기(032) / 보일러(033) / 순환펌프(034)

  035~038 통합: '장비 가동 현황' 패널 (작동상태 4종)
    CO₂공급(035) / 보광등(036) / 훈증기(037) / 보일러(038)

  차트 오버레이 활용:
    035(CO₂가동) → 022(CO₂농도) 차트 배경 밴드
    036(보광등가동) → 008(내부일사량) 차트 배경 밴드
    038(보일러가동) → 020(내부온도) 차트 배경 밴드
    039(주야간) → 모든 시계열 차트 야간 음영 기준값

  단독 불가 / 멀티라인 세트 구성 요소:
    040(환기온도) → 020(내부온도) + 040 + 041 세트

    - 027~034 전체
