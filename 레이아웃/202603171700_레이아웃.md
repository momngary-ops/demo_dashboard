# 레이아웃 확정 (2026-03-17)

## 전체 구조

```
┌──────────┬──────────────────────────────────┐
│  사이드바 │  헤더 (고정)                      │
│  (토글)  ├──────────────────────────────────┤
│          │  툴바 (페이지명 + 편집 액션)       │
│          ├──────────────────────────────────┤
│          │                                  │
│          │  그리드 영역 (react-grid-layout)  │
│          │  - 드래그 / 리사이즈 가능          │
│          │  - 위젯 추가 / 제거               │
│          │                                  │
└──────────┴──────────────────────────────────┘
```

## 사이드바

- **기본**: 200px 펼침 (아이콘 + 텍스트)
- **접힘**: 56px (아이콘만)
- **토글**: 좌측 상단 화살표 버튼 (`<` / `>`)
- 아이콘 및 메뉴명은 추후 수정 가능

## 헤더

- 높이: 56px, 고정
- 우측: 메일 아이콘 / 알림 아이콘 / 사용자 아바타 + 이름

## 대시보드 그리드

- 라이브러리: `react-grid-layout`
- 컬럼: 12
- 행 높이: 80px
- 마진: 12px
- **편집 모드** ON → 드래그 핸들 + 리사이즈 핸들 + 삭제 버튼 노출
- **위젯 추가**: 모달 피커에서 데이터 키 선택
- 위젯 타입: `stat` (수치 카드), `chart` (차트)
